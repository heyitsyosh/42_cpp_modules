<h1 align="center">
	C++ modules
</h1>

## Description:
Nine modules that introduce various C++ concepts through exercises.  
(Many of the exercises are based on fictional situations rather than practical problems.)  

## Overview:
| Module | Concepts covered |
|:---:|:---|
[**00**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module00#readme) | Namespaces, classes, member functions, stdio streams, static, const, other C++ basics |
[**01**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module01#readme) | Memory allocation, pointers to members, references, switch statement |
[**02**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module02#readme) | Ad-hoc polymorphism, operator overloading, Orthodox Canonical class form |
[**03**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module03#readme) | Inheritance |
[**04**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module04#readme) | Subtype polymorphism, abstract classes, interfaces |
[**05**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module05#readme) | Exceptions |
[**06**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module06#readme) | C++ casts |
[**07**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module07#readme) | C++ templates |
[**08**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module08#readme) | Templated containers, iterators, algorithms |
[**09**](https://github.com/heyitsyosh/42_cpp_modules/tree/master/module09#readme) | STL/containers |
